# fluxth's personal configuration files

Maybe unstable, use with caution!

## Requirements

- **Platform** 
  - Linux or macOS only
  - _Windows is not supported!_

- **Dependencies**
  - make
  - stow

## Installation

To install, run:
```bash
make
```

To uninstall, run:
```bash
make clean
```

Make will invoke GNU stow to create symlinks from this `dotfiles` repo to your XDG directories.
